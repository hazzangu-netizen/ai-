Action Note: 실행 관리형 AI 회의•팀플 노트
1. 앱 소개
Action Note는 회의나 팀 프로젝트 녹음 파일을 입력하면 AI가 내용을 분석해 핵심 요약부터 실행 가능한 할 일 목록까지 한 번에 정리해 주는 회의•팀플용 실행 관리 앱입니다.

Action Note는 단순히 대화 내용을 받아쓰는 일반적인 음성 인식 앱들과 다릅니다. Whisper Engine 기반의 고정밀 음성 인식을 통해 회의 내용을 텍스트로 변환하는 것은 물론, 대화 속 지시사항을 스스로 추론하여 담당자, 마감 기한, 우선순위가 포함된 액션 아이템(To-do)으로 즉시 바인딩해 주는 혁신적인 업무•학습 생산성 툴입니다.

2. 주요 기능
① Whisper Engine 기반 실시간 음성 분석
스마트 녹음 및 파일 업로드: 팀플, 업무, 프로젝트, 스터디 등 회의 목적에 맞는 템플릿을 선택한 후 실시간으로 녹음하거나 기존 오디오 파일을 업로드할 수 있습니다.

② Multi-stage 생성 추론 기반 리포트
구조화된 핵심 요약: 긴 회의 내용을 직관적인 '한 줄 브리핑'과 '3줄 맥락 요약'으로 압축하여 제공합니다.

③ To-do 자동 추출 및 담당자 주입
실행 중심 체크리스트 생성: 대화 속 지시사항과 흐름을 분석하여 담당자, 마감일, 우선순위(높음/중간/낮음)가 매핑된 할 일 목록을 자동으로 추출합니다.

④ 구글 캘린더 투인원(Two-in-One) 연동
원클릭 스케줄링: 추출된 마감일과 To-do를 별도의 번거로운 입력 없이 클릭 한 번으로 인-앱 타임라인 및 Google Calendar에 즉시 연동하고 상태 배지를 표시합니다.

⑤ 협업 전파 센터 & 아카이브
팀원 공유용 서식 자동 생성: 회의 요약, 결정 사항, To-do가 깔끔하게 정리된 공유용 텍스트를 생성하며 원클릭 복사 및 카카오톡, 이메일 공유 기능을 지원합니다.

3. 사용 방법
대학생들이 조별 과제(팀플) 회의를 마치고 다음 단계를 준비할 때, 사용자는 다음과 같은 흐름으로 앱을 실전 활용할 수 있습니다.

1단계: 회의 유형 선택 및 입력
앱을 켜고 [새 회의 녹음 시작] 또는 파일 업로드를 누른 뒤, 목적에 맞게 '팀플 회의' 템플릿을 선택합니다.

2단계: 회의 진행 및 녹음
팀원들과 자유롭게 의견을 나누며 회의를 진행합니다. AI가 실시간으로 음성 버퍼 세그먼트를 캡처합니다.

3단계: AI 실행 추론 연산 가동
회의가 끝난 뒤 [AI 분석 시작] 버튼을 누르면, Whisper와 LLM 오케스트레이션이 구동되며 다단계 추론을 통해 데이터를 가공합니다.

4단계: 분석 리포트 확인 및 To-do 바인딩
생성된 리포트에서 한 줄 요약과 결정 사항을 확인한 뒤, [내 To-do 탭으로 일괄 추가] 버튼을 눌러 팀원별 할 일 목록을 활성화합니다.

5단계: 캘린더 연동 및 팀 전파
[Google Calendar 연동] 버튼을 눌러 스케줄을 동기화하고, 자동으로 가공된 서식을 복사해 카카오톡 단톡방에 올림으로써 회의 결과를 즉시 전파합니다.

4. 사용자 가치
Action Note는 "기록에서 끝나는 회의록이 아니라, 행동으로 즉시 이어지는 혁신"이라는 슬로건 아래 다음과 같은 차별화된 가치를 전달합니다.

기록 전파의 리소스 제로화: 회의가 끝나자마자 팀원들에게 공유할 요약본과 역할 분담 텍스트가 완성되므로, 누군가 총대를 메고 회의록을 정리해야 하는 노동과 시간을 완전히 없애줍니다.

협업 효율 극대화 및 책임 명확화: 누가, 언제까지, 어떤 일을 해야 하는지 AI가 명시적으로 지정해 주므로, 팀플이나 업무 진행 과정에서 소통 오류로 인한 누락이나 마감 펑크를 방지합니다.

막연한 아카이브를 자산으로 변환: 지난 회의 기록이 깔끔한 카드형 UI와 실시간 검색/필터 시스템으로 누적되므로, 과거의 아이디어나 결정 사항을 필요할 때 언제든 다시 꺼내 쓸 수 있습니다.

높은 실행 몰입도: 투두리스트 작성, 일정 등록, 캘린더 연동 등 부가적인 관리에 드는 아까운 에너지를 터치 몇 번으로 단축시켜 사용자가 오직 '실제 행동'에만 몰입할 수 있도록 돕습니다.
